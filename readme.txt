Improve the Oxygen builder Editor: Structure opened by default, Back to Templates/Pages/Posts Menu, Display Currently Editing & more.

== Description ==

This plugin add some functionalities to the [Oxygen](https://oxygenbuilder.com/) Editor.

**Plugin Features:**

1. Oxygen Structure opened by default on editor loads.
2. Back to templates/pages menu in dropdown menu.
3. Display the current template/page name you are editing.
4. Editor UI improvements. 
 

There are no settings for the plugin. Just install it.




== Installation ==

=== Automatic Installation ===

Search for `better editor for oxygen` from within your WordPress plugins Add New page and install.

=== Manual Installation ===

1. Click on the `Download` button to download the plugin.
2. Upload the entire `better-editor-for-oxygen` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.
